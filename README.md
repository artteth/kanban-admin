## Инструкция по настройке Google Таблицы

### Шаг 1: Создайте новую Google Таблицу

1. Откройте https://sheets.google.com
2. Создайте новую таблицу
3. Назовите её, например, "Kanban Tasks v2"

### Шаг 2: Создайте лист "Пользователи"

1. В таблице создайте новый лист с именем `Пользователи`
2. Добавьте заголовки в первой строке:

| A | B | C | D |
|---|---|---|---|
| **UserID** | **Имя** | **TelegramID** | **Роль** |

3. Добавьте пользователей (пример):

| UserID | Имя | TelegramID | Роль |
|--------|-----|------------|------|
| user_1 | Артём | @artem | user |
| user_2 | Игорь | @igor | user |
| admin | Админ | @admin | admin |

### Шаг 3: Модифицируйте лист "задания"

В существующем листе `задания` добавьте новую колонку **H** с заголовком `AssignedTo`:

| A | B | C | D | E | F | G | **H** |
|---|---|---|---|---|---|---|---|
| **ID** | **Задание** | **Статус** | **Дата начала** | **Дата конца** | **Время выполнения** | **Плановая дата** | **AssignedTo** |

### Шаг 4: Скопируйте ID таблицы

После создания таблицы скопируйте её ID из URL:
```
https://docs.google.com/spreadsheets/d/1TsUjce91h44W_PF4dzCqCwGTB_jqhjJxRWBsLiGPmjE/edit
                                              ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
                                              ЭТО ВАШ ID ТАБЛИЦЫ
```

### Шаг 5: Настройте Google Apps Script

1. В таблице: **Расширения** → **Apps Script**
2. Откройте файл `Code.gs` (или создайте новый)
3. Скопируйте содержимое `cod_v2.gs` в этот файл
4. **Обновите константы:**
   - `SPREADSHEET_ID` - вставьте ваш ID таблицы
   - `SHEET_NAME_TASKS` - имя листа с задачами (обычно "задания")
   - `SHEET_NAME_USERS` - имя листа пользователей (обычно "Пользователи")
   - `TELEGRAM_BOT_TOKEN` - ваш токен бота

### Шаг 6: Разверните веб-приложение

1. В Apps Script: **Развернуть** → **Новое развёртывание**
2. Тип: **Веб-приложение**
3. Доступ: **Все пользователи** (или "Все, у кого есть аккаунт Google")
4. Нажмите **Развернуть**
5. Скопируйте URL веб-приложения

### Шаг 7: Настройте frontend

В файлах `select_user.html`, `index_v2.html`, `admin.html`:
- Найдите константу `API_URL`
- Замените на ваш URL из шага 6

### Шаг 8: Загрузите HTML файлы

Для Google Apps Script:
1. Создайте файлы в Apps Script: **select_user**, **index_v2**, **admin**
2. Скопируйте содержимое соответствующих HTML файлов
3. Или разместите на GitHub Pages / любом хостинге

## API Endpoints

### GET параметры

```
?mode=tg-api&action=getTasks&userId=USER_ID
?mode=tg-api&action=getUsers
```

### POST запросы (JSON)

```json
{
  "action": "addTask",
  "title": "Текст задачи",
  "plannedDate": "2026-03-05",
  "assignedTo": "user_1"
}
```

```json
{
  "action": "updateTask",
  "id": "task_123",
  "title": "Новый текст",
  "status": "in-progress",
  "plannedDate": "2026-03-06",
  "assignedTo": "user_2"
}
```

```json
{
  "action": "deleteTask",
  "id": "task_123"
}
```

```json
{
  "action": "addUser",
  "name": "Имя",
  "telegramId": "@telegram"
}
```

```json
{
  "action": "updateUser",
  "userId": "user_1",
  "name": "Новое имя",
  "telegramId": "@newtelegram"
}
```

```json
{
  "action": "deleteUser",
  "userId": "user_1"
}
```

## Роли пользователей

| Роль | Права |
|------|-------|
| **admin** | Все задачи, CRUD операций, управление пользователями |
| **user** | Только свои задачи, создание, редактирование своих |
