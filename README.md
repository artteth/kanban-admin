# 📋 Kanban Admin Panel v2

Система управления задачами с поддержкой нескольких пользователей.

---

## 🚀 Быстрый старт

### 1. Google Таблица

1. Создайте новую таблицу: https://sheets.google.com
2. Лист 1: `задания` с колонками:
   - ID, Задание, Статус, Дата начала, Дата конца, Время выполнения, Плановая дата, **AssignedTo**
3. Лист 2: `Пользователи` с колонками:
   - UserID, Имя, TelegramID, Роль

### 2. Backend

1. Откройте таблицу → **Расширения** → **Apps Script**
2. Вставьте код из `cod_v2.gs`
3. Обновите `SPREADSHEET_ID` в начале файла
4. **Развернуть** → **Новое развёртывание** → **Веб-приложение**
5. Доступ: **Все пользователи**
6. Скопируйте URL

### 3. Frontend

#### GitHub Pages:

1. **Settings** → **Pages**
2. **Source**: Deploy from a branch
3. **Branch**: main
4. **Save**

Через 1-2 минуты сайт доступен.

## 🐛 Решение проблем

### Ошибка: "JSONP request failed"
- Проверьте API_URL
- Проверьте права доступа к Apps Script

### Ошибка: "Пользователи не загружаются"
- Проверьте лист "Пользователи"
- Добавьте хотя бы одного пользователя
