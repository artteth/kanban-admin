# ⚡ Быстрый старт - Деплой за 15 минут

## 🎯 Краткая инструкция

### 1. Google Таблица (3 мин)

1. https://sheets.google.com → Создать
2. Лист 1: переименовать в `задания`
   - Заголовки: `ID`, `Задание`, `Статус`, `Дата начала`, `Дата конца`, `Время выполнения`, `Плановая дата`, `AssignedTo`
3. Лист 2: создать `Пользователи`
   - Заголовки: `UserID`, `Имя`, `TelegramID`, `Роль`
4. Скопировать ID таблицы из URL

### 2. Backend (5 мин)

1. В таблице: **Расширения** → **Apps Script**
2. Вставить код из `v2/cod_v2.gs`
3. Обновить `SPREADSHEET_ID` (строка 4)
4. Сохранить
5. **Развернуть** → **Новое развёртывание**
   - Тип: **Веб-приложение**
   - Доступ: **Все пользователи**
6. Скопировать URL

### 3. Frontend (5 мин)

**Вариант A: GitHub Pages**

```bash
# Копируем файлы
cp v2/select_user_local.html deploy/index.html
cp v2/admin_local.html deploy/admin.html
cp v2/index_local.html deploy/dashboard.html

# Обновляем API_URL в каждом файле
# Находим: const API_URL = '...'
# Заменяем на ваш URL из шага 2

# Загружаем
cd deploy
git init
git add .
git commit -m "Deploy"
git remote add origin https://github.com/НИК/РЕПО.git
git push -u origin main
```

**Вариант B: Netlify (проще)**

1. https://app.netlify.com/drop
2. Перетащите папку `deploy/`
3. Готово!

### 4. Проверка (2 мин)

1. Открыть сайт
2. Создать 3 пользователей: Админ (admin), Игорь (user), Мария (user)
3. Создать задачу
4. Проверить админку
