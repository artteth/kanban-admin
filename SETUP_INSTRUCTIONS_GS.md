# 📋 Kanban Board v2 - Инструкция по установке Google Sheets

## 🎯 Обзор

Эта версия Kanban-доски использует **Google Таблицы** для хранения данных вместо локального хранилища браузера. Все данные синхронизируются в реальном времени между всеми пользователями.

---

## 📁 Структура файлов

```
canban/
├── README.md                 # Общая информация
├── cod_v2.gs                 # Backend для Google Apps Script
├── select_user.html          # Страница выбора пользователя
├── index.html                # Kanban-доска для пользователей
├── admin.html                # Админская панель
└── SETUP_INSTRUCTIONS_GS.md  # Эта инструкция
```

## ⚙️ Пошаговая настройка

### Шаг 1: Создание Google Таблицы

1. Откройте https://sheets.google.com
2. Создайте новую таблицу
3. Назовите её "Kanban Tasks v2"

### Шаг 2: Настройка листов

#### Лист "задания"

Создайте лист с именем `задания` и заголовками:

| A | B | C | D | E | F | G | H |
|---|---|---|---|---|---|---|---|
| **ID** | **Задание** | **Статус** | **Дата начала** | **Дата конца** | **Время выполнения** | **Плановая дата** | **AssignedTo** |

**Важно:** Колонка H (AssignedTo) обязательна!

#### Лист "Пользователи"

| A | B | C | D |
|---|---|---|---|
| **UserID** | **Имя** | **TelegramID** | **Роль** |

### Шаг 3: Получение ID таблицы

Скопируйте ID из URL таблицы.

### Шаг 4: Настройка Google Apps Script

1. В таблице: **Расширения** → **Apps Script**
2. Скопируйте содержимое `cod_v2.gs`
3. Обновите константы с вашим ID таблицы

### Шаг 5: Развёртывание

1. **Развернуть** → **Новое развёртывание**
2. Тип: **Веб-приложение**
3. Доступ: **Все пользователи**
4. Скопируйте URL

### Шаг 6: Настройка frontend

Замените `API_URL` во всех HTML файлах:
- `select_user.html` (строка ~427)
- `index.html` (строка ~777)
- `admin.html` (строка ~857)
